### Usage:
```
git clone https://github.com/0xpsw/Instagram-Account-Generator.git
cd Instagram-Account-Generator
chmod +x instagen.sh
./instagen.sh
```

### Install requirements (Curl, Openssl):

```
apt-get install -y openssl curl
```
